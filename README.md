\# Apps Script Access Control (WebApp) — WIP



This project is an \*\*Apps Script–based web application\*\* that manages \*\*department-level access control\*\* for Google Sheets.  

The goal is to define and enforce \*\*which department can access which sheet\*\*, in a central, reusable way.



---



\## 📌 Project Status

\- \*\*In Development\*\*  

\- Currently exploring webapp architecture and how to apply access rules dynamically.  

\- Public repo intended for \*\*tracking development\*\* and \*\*receiving early feedback\*\*.  

\- A \*\*stable release\*\* with configuration examples and API docs will follow once core features are locked.



---



\## 🎯 Planned Features

\- Department → Sheet matrix (who can \*\*view\*\*, \*\*edit\*\*, or \*\*share\*\*).  

\- Webapp UI to manage rules without editing code.  

\- Central directory (map users → departments).  

\- Optional audit logging of all access attempts.  

\- Drive permission sync to enforce rules directly in Google Drive.
